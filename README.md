# A-To-Do-List-Application-Console-based-
This program is a menu-driven to-do list app. It lets the user:  View tasks stored in a list  Add new tasks  Remove tasks by number  Save all tasks in a file (tasks.txt) so they are not lost  Load tasks from the file when the program starts  It uses:  Lists to store tasks  File handling (open()) to save and load  Loops and conditions too.
